# 📅 Interactive Calendar App

✨ A beautifully designed **wall-style interactive calendar** built using React.js with smooth animations, modern UI, and real-world features like date range selection and holiday integration.

## ✨ Features

* 📆 **Dynamic Calendar Navigation** (Month switching with smooth UI)
* 🎯 **Date Range Selection** (Start → End with visual highlight)
* 🟢 **Today Indicator** (Current date always marked)
* 🔴 **Holiday Markers** (Fetched via API)
* 📝 **Notes Panel** (with localStorage support)
* 📱 **Fully Responsive Design** (Mobile + Desktop optimized)
* 🎨 **Aesthetic Wall Calendar UI** (Paper-style design with images)
* ⚡ **Fast & Optimized (Vite)**

---

## 🛠️ Tech Stack

* ⚛️ React.js
* ⚡ Vite
* 🎨 CSS (Custom Styling)
* 🌐 REST API (Holiday Data)
* 💾 localStorage
